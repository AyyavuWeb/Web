   → Done!

   → Save password securely!
   → Done!

================================================================================
DATABASE SCHEMA
================================================================================

projects table:
  - id (UUID)
  - title (text)
  - description (text)
  - details (text)
  - status (text: ongoing/completed/upcoming)
  - location (text)
  - price (text)
  - image (text)
  - created_at (timestamp)
  - updated_at (timestamp)

project_images table:
  - id (UUID)
  - project_id (UUID, foreign key)
  - image (text)
  - created_at (timestamp)
